### Configuration

Bukak-en file .env.development trus setup en dbmu dewe
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_DATABASE=anggal
DB_USERNAME=usernamemu
DB_PASSWORD=passmu
```
